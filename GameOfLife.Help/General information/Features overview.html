Game of Life

Features overview

  • General

    • Intuitive, user-friendly interface.
    • Work with clipboard.
    • Support hotkeys.
    • Support commands Undo/Redo for most operations.
    • Recording errors in the log file.
    • Installation and uninstallation.

  • Game

    • "Unlimited" size of the universe.
    • Coloring of living cells in different colors depending on its previous state.
    • Transformation of selected areas, such as rotations and flips.
    • Fitting and centering of the pattern to the screen.
    • Supports the following formats to load and save: RLE, Cells, Lif 1.05, Lif 1.06.
    • Various rules of the game: you can choose pre defined rules or set manually.
    • Saving screenshot of the field game in the file or clipboard.
    • Color settings: state of the cells, background, grid lines, etc.
    • Speed control generation of new generations.
