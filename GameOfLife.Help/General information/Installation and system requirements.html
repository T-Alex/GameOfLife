Game of Life

Installation and system requirements

Software requirements

  • 32bit or 64bit Windows XP/Vista, Windows 7.
  • .NET Framework 3.0 SP1 or higher.

Hardware requirements

  • 256 MB of RAM.
  • 10 MB of free hard disk space.

Installation

  • If you have installed an older version of this software, before starting the installation, be sure that the Game of Life is not running.
  • Run the installation of the new version and follow the instructions.

Uninstallation

  • Make sure that Game of Life is not running.
  • Run the "uninstall" from Add/Remove Programs of Control Panel.
