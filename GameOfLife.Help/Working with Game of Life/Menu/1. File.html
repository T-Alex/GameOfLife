Game of Life

Menu: File

  • New (Ctrl+N)
    Creates a new, empty universe. Scale is used by default, mode is set to draw.
  • Open... (Ctrl+O)
    Opens the standard file open dialog with which you can select a file pattern.
  • Save (Ctrl+S)
    Saves changes to current pattern file.
  • Save As...
    Opens the standard file save dialog with which you can save the current pattern in one of the possible formats.
  • Exit (Alt+F4)
    Closes the application.
