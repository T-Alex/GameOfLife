Game of Life

Menu: Edit

  • Undo (Ctrl+Z)
    Undoes the most recent undo command.
  • Redo (Ctrl+Y)
    Redoes the most recent undo command.
  • Cut (Ctrl+X)
    Copies to clipboard the selected pattern (in RLE format) and removes it from the field.
  • Copy (Ctrl+C)
    Copies to clipboard the selected pattern. The clipboard data is in RLE format.
  • Paste (Ctrl+V)
    Pastes pattern stored in the clipboard at one of the supported formats. To paste pattern in the desired location, click the desired area.
  • Delete (Del)
    Removes the selected pattern.
  • Select All (Ctrl+A)
    Selects the entire pattern, if there is at least one living cell.
  • Deselect All (Ctrl+D)
    Removes the current selection, if there is one.
  • Transform
    Different transformations of the selected area.
    • Rotate 180°
      Rotation of the selected pattern by 180 degrees.
    • Rotate 90° CW
      Rotation of the selected pattern by 90 degrees clockwise.
    • Rotate 90° CCW
      Rotation of the selected pattern by 90 degrees counterclockwise.
    • Flip Horizontal
      Flip Horizontal of the selected pattern.
    • Flip Vertical
      Flip Vertical of the selected pattern.
  • Paste mode
    Different modes of paste the copied patterns in the desired area.
    • Copy
      Logical function "OR" to paste pattern from clipboard into a previously cleaned area.
    • And
      Logical function "AND" to paste pattern from clipboard.
    • Or
      Logical function "OR" to paste pattern from clipboard.
    • Xor
      Logical function "XOR" to paste pattern from clipboard.
  • Cursor mode
    Three cursor mode for various editing operations.
    • Draw (F2)
      Mode drawing of pattern.
    • Move (F3)
      Mode moving on the plot.
    • Select (F4)
      Mode selecting of the pattern.
