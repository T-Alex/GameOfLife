Game of Life

Menu: Control

  • Start Game (Enter)
    Starts generating the current pattern.
  • Stop Game (Shift+Enter)
    Stops generating the current pattern.
  • Next Generation (Space)
    Advances the pattern to the next generation.
  • Reset Game (Ctrl+R)
    Restore pattern and generation count in the state before the first generation.
